# One Tool IT 홈페이지 개편 작업 요약

## 작업 배경

고객 니즈 조사 결과를 반영하여 홈페이지를 전면 개편함.

### 고객 니즈 우선순위
| 순위 | 니즈 | 설명 |
|------|------|------|
| #1 | 자산관리 + 실사 (엑셀 탈출) | 누가 쓰는지, 어디 있는지, 유휴·불용 자산이 뭔지 모르는 게 가장 큰 고통 |
| #2 | 입·퇴사 연계 자산 회수 + 유지보수 | 입퇴사 잦아지면서 자산이 잠기거나 고장 대응이 안 되는 문제 |
| #3 | 데이터 완전 삭제·보안 인증 | 금융/보안 민감 기업에서 중요하지만 전체로 보면 선택적 니즈 |

### 핵심 컨셉
- **기존**: "IT 기기 렌탈/판매 업체" 이미지 (공급자 중심)
- **변경**: "IT 자산관리 솔루션 + 거래 원스톱" (문제 해결 중심)
- **수익 모델**: 관리 솔루션(무료)으로 락인 → 구매/매각/렌탈에서 수익 창출

### 1. HeroSection.tsx
**경로**: `src/components/home/HeroSection.tsx`

#### 변경 전

#### 변경 후

#### 주요 변경사항
- 메인 카피: 고객의 Pain Point(엑셀 관리) 직접 언급
- 서브 카피: "관리 + 거래 원스톱" 메시지
- 체크리스트:
  - ✓ 자산 현황 실시간 파악 + 거래 타이밍 알림
  - ✓ 입퇴사 자산 회수 → 매각/재배치 원클릭 연결
  - ✓ 교체 적기 알림 → 리퍼비시 30% 절감 견적
  - ✓ 관리 솔루션 무료 제공, 거래까지 원스톱
- CTA 버튼: "무료 자산 현황 진단받기", "우리 회사 절감액 계산하기"
- 대시보드 목업: 자산현황/입퇴사연계/유휴자산 카드로 변경
- **스마트 거래 알림 추가**:
  - 노트북 3대 매각 적기 → 예상 120만원
  - 모니터 5대 교체 추천 → 리퍼 30% 절감
  - 퇴사자 장비 회수 완료 → 재배치/매각 선택

## 핵심 메시지 정리

### Before (공급자 중심)
> "B2B IT 디바이스 종합 서비스 - 매각·구매·렌탈·유지보수·자산실사"

### After (고객 중심 + 거래 연결)
> "아직도 엑셀로 자산 관리하시나요? 관리부터 구매·매각·렌탈까지, 하나의 플랫폼에서"

### 핵심 차별점
1. **관리 솔루션이 단순 관리가 아닌 "거래 타이밍 알림 도구"**
   - "이 노트북 3대 매각하면 120만원"
   - "이 모니터 5대 교체 적기, 리퍼비시로 30% 절감"

2. **입퇴사 자산 회수 → 거래 연결**
   - 퇴사자 장비 회수 완료 → "매각 or 재배치" 선택 화면

3. **무료 관리 솔루션으로 락인 → 수익은 거래에서**
   - 관리 솔루션: 무료 (고객 유입 & 락인)
   - 구매/매각/렌탈: 수익 창출
